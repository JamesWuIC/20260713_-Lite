# 洲遊幣 LINE 活動｜個人資料蒐集同意條款（草稿）

> 版本：v0.1（供內部與法務討論用）
> 適用情境：LINE 小遊戲 + 抽獎，兩家酒店各自為蒐集者、使用者可分別勾選。

## 〇、給法務／內部的重點提示（不對外顯示）

1. **兩家為各自獨立蒐集者**：使用者勾哪家，哪家才取得資料；未勾選之酒店「完全不取得」該筆資料。本設計「不涉及提供第三人」，惟兩份告知與同意須各自獨立、可單獨勾選。
2. **自由同意 + 可見替代路徑**：抽獎不以「同意行銷」為必要條件。取得洲遊幣的方式須「兩條並列、公開可見」（見第四節抽獎規則），字級／位置可次要，但不得隱匿。
3. **委外與國際傳輸（已確認：Supabase 官方託管）**：本次採 Supabase 官方託管，Supabase Inc.（及其底層 AWS）為受託處理者，須與 Supabase 簽署／接受其標準 DPA（資料處理協議）並盡監督義務（個資法施行細則第 8 條）。Supabase 無台灣機房，建議建立專案時即選 **Tokyo（ap-northeast-1）**（region 於建立時決定、事後不可變更）；資料存於日本，屬**國際傳輸**，告知本文「利用地區」已據實載明。另請資安確認已對含個資之資料表啟用 RLS（Row Level Security）。
4. **保存期間**：「長期會員行銷池」≠ 無限期。建議設「至撤回同意／要求刪除時止，並定期（如每 3 年）檢視」，避免無期限保存之風險。
5. **未成年**：既有觸及可能，建議活動頁明載「限 18 歲以上」＋年齡確認勾選；如允許未成年參加，須另設法定代理人同意機制。
6. **撤回須可分店獨立執行**：系統面請預留 `consent_status`、`consent_withdrawn_at` 等欄位，確保「撤回」是真的能停止推播，而非僅條款文字。
7. **稅務**：抽獎獎項涉「機會中獎所得」之列單與代扣繳義務，請洽財務主管確認，本草稿不載具體金額與稅率。

**【蒐集者】**
本活動由下列二家公司各自為蒐集者，各自獨立蒐集、處理及利用您所提供之個人資料。您同意提供予哪一家，該公司始取得您的資料；未經您勾選之公司，不會取得您的個人資料。

- 臺北洲際酒店【公司全名：遠雄巨蛋事業股份有限公司逸仙酒店分公司】【統一編號：60613318】

**【蒐集目的】**（特定目的代號供法務核對）
會員管理（052）、消費者／客戶管理與服務（090）、行銷（040）、調查統計與研究分析（157，含會員分析、貼標與個人化推薦）。

**【蒐集之個人資料類別】**
辨識個人者（C001）：姓名、行動電話、電子郵件；及您透過 LINE 登入所提供之 LINE 使用者識別碼（userID）、LINE 顯示名稱。

**【利用期間】**
自您同意時起，至您撤回同意、請求刪除，或該蒐集之酒店停止相關營業時止（本公司並將定期檢視保存必要性）。

**【利用地區】**
中華民國境內，及本公司個人資料實際儲存與處理所在之地區。本活動使用外部雲端服務 Supabase（其伺服器位於**日本（東京）**）儲存與處理資料，即涉及國際傳輸；您同意本告知範圍內之利用。

**【利用對象】**
- 您所勾選同意之酒店本身；
- 為提供前述服務所必要之受託處理者（本活動之雲端資料庫服務商 Supabase 及其底層雲端業者等），本公司將對其盡監督義務。
- ※ 未經您勾選之另一家酒店，不屬利用對象，不會取得您的資料。

**【利用方式】**
以電子郵件（EDM）、簡訊、LINE 推播、電話等方式進行活動聯繫與行銷；並可能以自動化方式進行會員分析、貼標及個人化推薦（profiling）。

**【您得行使之權利（個資法第 3 條）】**
您得向蒐集之酒店請求：查詢、閱覽、製給複製本、補充或更正、停止蒐集處理或利用、刪除，並得隨時撤回同意。上述權利不因本同意而預先拋棄，行使方式詳見第三節。

**【不提供或不同意之影響】**
提供個人資料及是否同意行銷，均由您自由選擇。**若您不同意行銷，仍可參加本活動及抽獎**——您可透過活動規則所列之其他方式取得洲遊幣（見活動規則）。不同意之影響僅為：無法以「填資料領幣」方式取得該枚洲遊幣，且不會收到本公司行銷資訊。

## 三、查詢／更正／停止利用（撤回）管道

> 首次行銷利用時即應提供拒絕方式，且相關費用由本公司負擔。

1. **客服信箱／表單**：如需查詢、閱覽、複製、更正、停止利用、刪除或撤回同意，請來信【privacy 信箱：待填】或填寫【線上表單連結：待填】。我們將於法定期間內處理，且不向您收取費用。
2. **行銷管道內建退出**：每封 EDM 均附退訂連結；簡訊可回覆指定關鍵字退出；LINE 推播可依指示退出或封鎖。
3. **分店獨立撤回**：因兩家酒店各自蒐集，您可僅就其中一家撤回同意而不影響另一家；請於來信時載明對象。
